# Bhapi Social — Release Notes

## v4.0.0 (2026-03-24)

### What's New

**Safe Social Network Launch**
Bhapi Social is now available — a parent-supervised social network built for children and teens. Every connection requires parental approval and all content passes age-appropriate moderation before it is visible.

**Art Studio with AI Generation**
Children can describe a scene in plain language and the Art Studio generates safe, age-appropriate images using a moderated AI pipeline. All generated images are queued for parental review before sharing.

**Story Creator with Templates**
A library of story templates (adventure, mystery, nature, science) lets kids build illustrated short stories with guided prompts. Completed stories can be shared with approved contacts or kept private.

**Drawing Canvas**
A full-featured drawing tool with brushes, stamps, and colour palettes is built into the app. Drawings are saved to the child's gallery and can be shared or submitted for Creative Tools review by parents.

**Sticker Packs**
Curated sticker collections — including seasonal, animal, and space themes — are available for use in messages and stories. New packs are reviewed before release and rotate monthly.

**Parent-Approved Contacts**
Children can send connection requests, but both the recipient and a parent on each account must approve before messaging opens. Parents receive a notification for every pending request and can revoke any connection at any time.

**Age-Appropriate Content Moderation**
All text, images, and stories pass a multi-layer moderation pipeline tuned for the platform's age range. Content that cannot be automatically cleared is held for human review before delivery.

### Bug Fixes & Improvements

- Story save progress is now preserved if the app is backgrounded mid-edit
- Drawing canvas performance improved on lower-end Android devices
- Contact request notifications now group by sender to reduce notification fatigue
- Improved accessibility labels on all interactive elements (WCAG 2.1 AA)
